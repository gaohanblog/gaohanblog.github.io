美团校招笔试题-字符串模式匹配

发表于 2020-01-22 分类于 数据结构与算法 ， 笔试题
本文字数： 2.2k 阅读时长 ≈ 2 分钟

题目描述

给出两个字符串，分别是模式串P和目标串T，判断模式串和目标串是否匹配，匹配输出 1，不匹配输出 0。模式串中‘？’可以匹配目标串中的任何字符，模式串中的 ’*’可以匹配目标串中的任何长度的串，模式串的其它字符必须和目标串的字符匹配。例如P=a?b，T=acb，则P 和 T 匹配。

输入描述:

  • 输入第一行包含一个字符串p， (1≤|p|≤20).
  • 输入第二行包含一个字符串t， (1≤|t|≤20).

输出描述:

  • 输出仅包含0和1的整数，0表示p和t不匹配，1表示p和t匹配。

示例1

输入:

s1 = a?b

s2 = ab

输出:0

示例2

输入:

s1 = a*b

s2 = ab

输出:1

示例3

输入:

s1 = a*b

s2 = a(cb

输出:
1

解题思路

这个题目和leetcode第10题有很多相似的地方，但也不完全行同。leetcode第十题对于*的描述是 匹配零个或多个前面的那一个元素，这个题目对于*的描述是可以匹配目标串中的任何长度的串。所以类似的题目也可以用类似的思路来所。

和第10题一样定义dp的状态为二维数组的dp[i][j] 表示目标串的前 i 个是否能被模式串的前 j个字符匹配。那么状态转移方程可以分两种情况讨论：

  • 第一种假如当前被遍历的s1的第i个字符和s2的第j个字符相等，或者s1的字符为?,说明这两个字符是可以匹配上的，所以他们的结果应该由前一个状态来决定，即dp[i][j] = dp[i - 1][j - 1]
  • 第二种就是假如被遍历的s1的字符为*,那么它可以匹配0个或无限个字符。
    • 当匹配0个的时候，例如s1 = [aa*] s2 = [aa],他的状态由i-1决定即dp[i][j] = dp[i - 1][j]
    • 当匹配多个的时候，例如s1 = [aa*] s2 = [aabcd]那么他应该由j - 1决定（无论s2后面有多少个字符都可以匹配上），即dp[i][j] = dp[i][j - 1]
    • 所以dp[i][j] = dp[i - 1][j] || dp[i][j - 1]
