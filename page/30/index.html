题目描述

输入一个链表，输出该链表中倒数第k个结点。

解题思路

因为找到链表中倒数第k个节点，定义两个快慢指针，先让一个指针走k个节点，然后再让两个指针一块走，当先走的指针到尾节点时，后面的指针和他相距k，即指向的是倒数第k个节点

【剑指offer 09-变态跳台阶】

发表于 2019-04-23 分类于 数据结构与算法 ， 剑指offer
本文字数： 599 阅读时长 ≈ 1 分钟

题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

解题思路

上个问题中青蛙一次只能跳一阶或者两阶 这次可以跳n阶
所以

  • 当只有一个台阶 f(1) = 1;
  • 两个台阶 f(2) = f(2-1) + f(2-2) f(2-2)表示2阶一次跳两阶的次数
  • 三个台阶 f(3) = f(3-1) + f(3-2) + f(3-3) n=3时青蛙有三种跳的方式 当第一次跳一阶那么剩下f(3-1),第一次跳两阶剩下f(3-2),第一次跳三阶剩下f(3-3)
  • 同理当为n阶梯时 f(n) = f(n-1) + f(n-2) + f(n-3)+…+f(n-n)
    • 根据上个式子 f(n-1) = f(n-2) + f(n-3) +…+f(0)
    • 可以推导f(n) = 2 * f(n-1)

【剑指offer 08-跳台阶】

发表于 2019-04-23 分类于 数据结构与算法 ， 剑指offer
本文字数： 520 阅读时长 ≈ 1 分钟

题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

解题思路

因为青蛙一次只能跳一个或者两个台阶

  • 跳一个台阶只有一种跳法 即f(1) = 1 跳二个台阶有 两种跳法 即f(2) = 2
  • 如果第一次青蛙跳一个台阶，那么剩下的台阶有f(n-1)种跳法
  • 如果青蛙第一次跳两个台阶，那么剩下的台阶有f(n-2)中跳法
  • 可以推导出 f(n) = f(n-1) + f(n-2);

【剑指offer 06-旋转数组的最小数字】

发表于 2019-04-23 分类于 数据结构与算法 ， 剑指offer
本文字数： 863 阅读时长 ≈ 1 分钟

题目描述

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。 输入一个非减排序的数组的一个旋转，输出旋转数组的最小元素。 例如数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转，该数组的最小值为1。 NOTE：给出的所有元素都大于0，若数组大小为0，请返回0。

解题思路

  • 因为是非减数组这题最直接的思路是先从头遍历找到分界点(分界点是找到一个数比前面的小)返回这个分界点的数即可，复杂度O(n)
  • 可以采用二分查找的方法降低时间的复杂度为O(log(n))，下面的方法就是二分查找。
    • 先定义找到中间值mid最后一个值end比较他们两个的大小 如果中间值大则target在mid的后面，如果小则在前面
    • 假如说相同，则让end–比如 [1,1,0,1,1,1]只能一个一个去试。

【剑指offer 05-用两个栈实现队列】

发表于 2019-04-23 更新于 2019-07-17 分类于 数据结构与算法 ， 剑指offer
本文字数： 754 阅读时长 ≈ 1 分钟

题目描述

用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。

解题思路

队列特点先进先出，栈是后进先出。假如只用一个栈来操作pop()和push()肯定是不行的 。我们可以把stack1用来装元素，stack2向外出元素 假如stack2中元素为空，那么我们可以直接把stack1所有的元素倒进stack2中这样就实现了先进先出的原则。
