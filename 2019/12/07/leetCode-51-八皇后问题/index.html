leetCode-51-八皇后问题

发表于 2019-12-07 分类于 数据结构与算法 ， leetcode
本文字数： 2.8k 阅读时长 ≈ 3 分钟

题目描述

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。

示例:

输入: 4

输出: [

[“.Q..”, // 解法 1

“…Q”,

“Q…”,

“..Q.”],

[“..Q.”, // 解法 2

“Q…”,

“…Q”,

“.Q..”]

]

解释: 4 皇后问题存在两个不同的解法。

解题思路

整理下回溯法整体的思路：

  • n*n的棋盘，当第一个棋子放在棋盘第一行第一列，则棋盘的第一列被攻击，当前棋子的格子到右下角被攻击
  • 第二个棋子，从左到右检查第二行不能被攻击的格子，放下棋子。
  • 第三个，和第二个棋子一样检查第三行不被上面两个棋子攻击的格子。
  • 假如第n个棋子他的第n行都被前面的攻击，说明前面棋子放的不可能有正确的结果集。那么回溯到上一层，继续向当前行后面遍历找到可以放的地方。
  • 假如我们遍历到最后一层找到可以放的格子，则保存到结果集。
  • 继续遍历，直到找不到不被攻击的格子，回溯到上一层。
  • 直到遍历玩每一行的最后一个元素，结束程序。

分析下时间的复杂度，放第一个皇后有N种放法，放第二个皇后不超过N(N-2),第三个皇后不超过N(N-2)(N-4)种放法，以此类推。总体上，时间复杂度为N的阶乘O(N!)。
