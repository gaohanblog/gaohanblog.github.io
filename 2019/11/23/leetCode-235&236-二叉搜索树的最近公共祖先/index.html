leetCode-235&236-二叉搜索树的最近公共祖先

发表于 2019-11-23 分类于 数据结构与算法 ， leetcode
本文字数： 1.9k 阅读时长 ≈ 2 分钟

题目描述

给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

例如，给定如下二叉搜索树: root = [6,2,8,0,4,7,9,null,null,3,5]

示例1 ：

输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = >2, q = 8

输出: 6

解释: 节点 2 和节点 8 的最近公共祖先是 6。

示例 2:

输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4

输出: 2

解释: 节点 2 和节点 4 的最近公共祖先是 2, 因为根据定义最近公共祖先节点可以为节点本身。

236题和235不同的是，一个给定的是二叉搜索树，一个给定的是二叉树。

解题思路

这个题目要求找到两个节点公共祖先，那么最直接的思路就是顺着两个节点的父节点找，第一个重合的部分就是公共祖先，但是题目中并没有子节点指向父节点的指针，仅代表一种最直接思路。

利用二叉搜索树的性质，直接比较当前节的值和p节点值和q节点值，如果p大于当前节点则在右子树，反之一样，假如相同直接返回当前节点即可。

leetcode236，给定的是一个二叉树，也就是不可以用比较节点的值，这时候用递归递归遍历找到节点层层返回找到节点的值。
